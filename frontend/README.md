# Entropy Civil – Frontend

React + Vite + Three.js で構築されたリアルタイム文明観測 UI。

## Development

```bash
npm install
npm run dev       # http://localhost:5173
```

## Environment

`.env` に以下を設定:

```env
VITE_API_BASE_URL=http://localhost:8001
```

See root [README.md](../README.md) for full project setup.
